# Evaluation Datasets Overview

78 evaluation subsets across 3 modalities, all using Lance three-file layout (queries / candidates / qrels).

**Metric:** hit@1 (image/video), ndcg@5 (visdoc)

---

## Modalities

| Modality | Subsets | Task Types |
|----------|---------|------------|
| [Image](./image/index.md) | 36 | Classification, VQA, Retrieval, Visual Grounding |
| [Video](./video/index.md) | 18 | Classification, QA, Retrieval, Moment Retrieval |
| [VisDoc](./visdoc/index.md) | 24 | ViDoRe V1, ViDoRe V2, VisRAG, OOD |
