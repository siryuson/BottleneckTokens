# VOC2007

> PASCAL VOC 目标类别识别子集，用于通用视觉分类评测。

| 属性 | 值 |
|------|-----|
| 模态 | image |
| 任务类型 | `image_classification` |
| 用途 | 评测 |
| 溯源层数 | 3 层 |

## 原始数据集
### 来源信息
| 属性 | 值 |
|------|-----|
| 原始名称 | PASCAL VOC 2007 |
| 论文 | The PASCAL Visual Object Classes (VOC) Challenge ([arXiv](https://arxiv.org/abs/0909.5206)) |
| 主要作者 | Mark Everingham, Luc Van Gool, Christopher K. I. Williams, John Winn, Andrew Zisserman |
| HuggingFace | [merve/voc2007](https://huggingface.co/datasets/merve/voc2007) |
| 许可证 | 参见原始数据集 |

### Schema
图像 + 类别标签。

### 样本

**query.text**: `<|image_pad|>...`
**query.images**: 1 张图像
**candidate.text**: `aeroplane...`
**candidate.images**: 0 张图像

## BToks Lance 格式
### 来源信息
| 属性 | 值 |
|------|-----|
| 目录 | `MMEB-V2/image-tasks/VOC2007/` |
| task_type | `image_classification` |
| 评测模式 | local |

### Lance 布局
`queries.lance` / `candidates.lance` / `qrels.lance` / `metadata.json`。

### 样本

以下示例直接采样自 `./data/converted`。

- `text` 字段保留 Lance 原值。
- `images` 为二进制列，文档中按 `count / format / size / bytes / sha1` 展示摘要，不直接展开原始 bytes。

#### 示例 1
`query_id=q_0`，`positive_candidate_id=c_8`

```json
{
  "queries.lance": {
    "id": "q_0",
    "text": "<|image_pad|>\nIdentify the object shown in the image\n",
    "images": {
      "count": 1,
      "items": [
        {
          "index": 0,
          "bytes": 1752,
          "sha1": "0713cd2bf3d0",
          "format": "JPEG",
          "size": [
            49,
            85
          ]
        }
      ]
    }
  },
  "candidates.lance": {
    "positive": {
      "id": "c_8",
      "text": "chair",
      "images": {
        "count": 0,
        "items": []
      },
      "score": 1.0
    }
  },
  "qrels.lance": {
    "query_id": "q_0",
    "mode": "sparse",
    "positive_candidate_ids": [
      "c_8"
    ],
    "positive_candidate_scores": [
      1.0
    ],
    "total_candidate_ids": 1,
    "negative_candidate_count": 0
  }
}
```
